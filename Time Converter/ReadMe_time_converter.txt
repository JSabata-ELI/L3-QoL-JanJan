Do you want to convert timestamp in the name of files to readable form in our new archiver?

Select If you want to convert it into Prague Time and if you want detailed output log.
Click on COPY button and select files you want to convert.
After that select folder you want to copy them in.
	. If you select the detailed output log you will see 1 window with confirmation and what will be done and second window with what was done.

Files that are already in the final format will be automatically skipped.
Files without a UNIX time will be skipped.
Existing files in the target folder with the same name will be overwritten.

Estimated time of conversion - 10k files in about 1m.
