This program purpose is to have all of the programs we have created in one place.

If you are using this program from the office, you should first set up the path for Scratch or Sharepoint folder.
	. Click on "Set Path" in the right upper corner.
	. Choose the path you want to set and set it according to the text in the program.
		.. For SharePoint you need to select Documents\General\QoL folder; for Scratch you need to set Scratch\Software folder.

If you are in the lab, you don't have to set anything. You can just use the "Lab - Scratch" path.

Click on the station you are in.

Click on program you want to run and wait a little bit.

There is even information about each program in "readme" button.

"Notes" button opens "notes.txt" on Scratch\Software path.
	. You need to have chosen station.
